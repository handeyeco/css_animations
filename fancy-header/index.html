Two Elements, One Gradient

This seems like the most effective method because the line resizes with the content. Add a gradient on the background div that makes the line and then overlay an element with a background that matches the main background. Unfortunately this means when the main background is changed, the top element also needs its background changed.

One Element, Two Gradients

This uses two gradients, one to create the line and another to white out the line over the content. This doesn't resize with the content and must be changed when the background changes. Worst method.

One Element, One Pseudo Ele

This is a good method in that it's transparent, so colors don't need to be adjusted. The downfall of this method is that it's a little weird (using pseudo elements) and doesn't adjust with content. It's also a little more work placing the psuedo elements.
